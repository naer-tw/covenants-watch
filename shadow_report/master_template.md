# 兩公約施行總檢討民間影子報告（草案 v0.1）

> 本報告為**骨架版**，待 Wave 21+ 實證資料匯入後逐步充實。
> 上線發布前須通過外部審查（含反對立場學者）+ 法律顧問複核。

## 執行摘要

兩公約於 2009 年通過《兩公約施行法》，至 2026 年施行 16 年。本報告以雙軸框架審視：

**軸一**：四次審查 250+ 條結論性意見之執行落差
**軸二**：條文援引、立法院公報、白皮書、教科書等公領域之客觀檢驗

本報告**不採單向倡議文宣**之體例，採學術研究 + 雙面 Steelman + 三公約對照組之嚴謹方法。

## 第一章 平台立場與方法論

> 詳見 `_public/about.html` 與 `_public/methodology.html`

### 1.1 平台立場（紅線）
1. 支持兩公約原始普世人權精神
2. 反對工具化（單一議程擋箭牌）
3. 反對政黨化（選擇性援引）
4. 客觀證據優先 / 對事不對人

### 1.2 雙軸框架
- 類別一：過去要求沒兌現
- 類別二：援引行為與政策落差客觀檢驗

### 1.3 三公約對照組（防止單獨歸因）
- CRC、CEDAW、CRPD 同步跑相同手法
- 兩公約偏差須 > 對照組中位數才能下「特別工具化」結論

### 1.4 雙面 Steelman 規範
- 每章必含對立論述之最強版本
- PI-10 採對等雙面 Steelman 架構
