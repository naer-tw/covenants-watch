# 兩公約議題倡議速查包（家長團體 / 教會 / NGO 用）

> 更新:2026-05-04｜涵蓋:16 張 PI 卡

## 一、最常被誤用的 4 個法律邏輯(辨識指引)
1. **援引兩公約但未標 GC 段落號** — 主張「兩公約要求 X」之文宣,若未標明 General Comment 段號,通常為「援引者個人詮釋」,非條約原文。
2. **將條文 A 用來吞掉條文 B** — 例如以 ICCPR §26(不歧視)凌駕於 ICCPR §18(宗教自由),但二者為同等位階,聯合國人權事務委員會 GC22 §11 明示宗教自由之核心不可被「平衡」。
3. **混淆「公約原文」與「一般性意見」** — GC 由委員會制定,屬詮釋指引而非條約文,各國司法機關可拒絕跟隨;台灣司法院亦未具拘束力之裁定。
4. **以「進步詮釋」覆蓋父母教育選擇權** — ICESCR §13-3 明文保障父母依其信念選擇宗教與道德教育,GC13 §28 進一步強調此為核心權利。

## 二、5 個立場明確之關鍵條文(附 GC + 段號)
- **ICCPR §18** 思想良心宗教自由 — GC22(1993),特別 §3、§8、§11 之核心保障
- **ICCPR §19** 言論自由 — GC34(2011),三要件:法律明確、合法目的、必要與比例
- **ICCPR §22** 結社自由 — GC10 + 工會權之最低標準
- **ICCPR §26** 不歧視 — GC18(1989),平等保護但不延伸到強制信念之同質化
- **ICESCR §13-3** 父母教育選擇權 — GC13(1999)§28、§30,含「依其信念之宗教與道德教育」

## 三、可援引之國際判例(對照組)
- **Hosanna-Tabor v EEOC**(565 U.S. 171, 2012)— 美國最高法院確立宗教團體之雇用自治(ministerial exception)
- **303 Creative LLC v Elenis**(600 U.S. 570, 2023)— 言論自由凌駕反歧視法之具體應用
- **Mahmoud v Taylor**(605 U.S., 2025-06-27)— 父母依宗教信念請求子女不參與特定課程之保護
- **Eweida v UK**(ECHR 48420/10, 2013)— 工作場所宗教表達之歐洲標準
- **Lautsi v Italy**(ECHR 30814/06, 2011 GC)— 公立學校懸掛宗教象徵之合法性

## 四、雙面 Steelman 反論模板
當對方主張「條文 X 要求 Y」時,可依下列順序回應:
1. **要求 GC 段號** — 「請問是哪一條 General Comment 之第幾段?」(若對方答不出,主張站不住腳)
2. **回援同位階條文** — 例如對方援 §26,你可援 §18 並指出 GC22 §11 之核心保障
3. **指出國際判例反例** — 例如援 Hosanna-Tabor 反例平衡性過度延伸
4. **要求平衡執行** — 「兩公約 22 條核心條文應均衡執行,而非選擇性援引」
