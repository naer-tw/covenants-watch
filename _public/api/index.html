6 個 endpoints

GET /api/index.json
API 元資訊 + 所有 endpoint 清單 + schema 文件 + 使用範例
~ 2.4 KB
GET /api/pi.json
16 PI 議題卡 metadata(pi_id / title / status / covenant / keywords / file_path)
~ 7 KB · 16 items
GET /api/trace.json
因果脈絡追溯系統(actor 107 + event 100 + causal_link 74 + outcome_indicator 33)
~ 163 KB · 314 records
GET /api/laws.json
6 部核心法律修法歷程(law + version + article_change + amendment + executive_order)
~ 36 KB · 53 records
GET /api/timeline.json
混合 events 與 amendments 之全平台時序軸(統一 schema 含 source 欄)
~ 43 KB · ~107 items
GET /api/coverage.json
PI 狀態 + 法律覆蓋 + 統計總覽(用於儀表板 dashboard)
~ 4 KB
GET /api/search-index.json
fuse.js 全文搜尋索引(已部署於 search.html)
~ 98 KB · 278 items

使用範例(JavaScript)

// 1. 抓所有 PI
const pi = await fetch('/api/pi.json').then(r => r.json());
console.log(`共 ${pi.items.length} 張議題卡`);

// 2. 過濾廢死相關事件
const trace = await fetch('/api/trace.json').then(r => r.json());
const deathPenalty = trace.events.filter(e =>
  e.issue_tags && e.issue_tags.includes('廢死')
);

// 3. 找出 ICCPR §18 相關之法律條文 delta
const laws = await fetch('/api/laws.json').then(r => r.json());
const art18changes = laws.article_changes.filter(c =>
  c.related_article && c.related_article.includes('ICCPR-18')
);

// 4. 串接 causal_link 與 events
const linkChain = trace.causal_links.map(link => ({
  ...link,
  from: trace.events.find(e => e.event_id === link.from_event),
  to: trace.events.find(e => e.event_id === link.to_event),
}));

使用範例(Python)

import requests

base = 'https://covenants.aabe.org.tw/api'  # 部署後
# 或本地: http://localhost:8765/api

trace = requests.get(f'{base}/trace.json').json()
print(f"actors: {len(trace['actors'])}")
print(f"events: {len(trace['events'])}")

# 篩選 2024 年後之事件
recent = [e for e in trace['events'] if e['event_date'] >= '2024']
print(f"2024+ 事件:{len(recent)} 筆")

Schema 摘要

表主要欄位
actoractor_id, actor_type, name, affiliation, position_spectrum, active_period, is_anonymized
eventevent_id, event_date, event_type, issue_tags, actor_id, title, summary, related_pi, related_co, related_article, is_positive_outcome
causal_linkfrom_event, to_event, link_type, evidence_strength, chain_depth, counter_evidence, multi_causal_note
outcome_indicatorindicator_id, event_id, metric_name, before_value, after_value, direction, confounders
lawlaw_id, law_name_zh, law_pcode, enacted_date, primary_agency, covers_articles
law_article_changechange_id, version_id, article_number, change_type, text_before, text_after, reason, related_co, related_article
