回饋與訂正

本平台所有內容歡迎挑戰。若您發現引用錯誤、條文誤讀、量化方法錯誤、或您的組織 / 立場被錯誤呈現，請循以下管道通報。

本表單功能於 2026 年下半年正式上線時啟用（將整合 Tally 表單）。 目前先提供 email 與 GitHub Issue 通道。

一、四類回饋管道

A. 引用錯誤訂正

若您發現本平台引用之條文號 / CO 段落號 / 統計數字 / 來源 URL 有誤，請提供：

  • 錯誤所在頁面（URL）與段落號
  • 原文錯誤內容
  • 您主張之正確內容
  • 支持您主張之原始出處（URL 或文件名）

本平台承諾 72 小時內 回覆，必要時公開訂正並更新版本號。

B. 立場分類異議

若您是本平台引用之 NGO / 政黨 / 學者，認為您的立場被錯誤分類，請提供：

  • 本平台對您之分類描述（請貼具體段落）
  • 您主張之正確立場（請附您組織之公開官方文件 URL）
  • 是否同意本平台公開引用您之澄清

本平台承諾 14 日異議期，異議期內不發布相關量化結果；成立則修正。

C. 方法論挑戰

若您是研究方法專業者，認為本平台之研究設計（如 chi-square 之適用性、Control Group 是否充分、Steelman 強度）有問題，請提供：

  • 挑戰之具體章節 / 議題卡編號
  • 您主張之方法論問題
  • 建議修正方向
  • 支持您論述之學術文獻

本平台承諾 30 日 內公開回應；若挑戰成立則修訂研究設計並記錄於 DECISIONS.md（ADR）。

D. 外部審查邀請

本平台主動邀請以下三類外部審查者：

  • 研究方法論專業者（統計或公共政策博士）
  • 法律學者（人權法 / 憲法 / 國際法）
  • 反對本平台立場之學者（必要邀請對象）

若您願意擔任外部審查者，請聯繫本平台。審查意見摘要將於對應發布頁面公開。

二、聯絡通道

  • Email：透過 aabe.org.tw 之聯絡頁面
  • GitHub Issue：本平台原始碼倉庫之 issues 區（部署上線後公告）
  • 正式公文：寄送至國教行動聯盟（地址依官網公告）

三、本平台回應原則

  • 所有回饋一視同仁處理，不依立場區別
  • 所有回應公開存檔（除涉個資 / 法律敏感案件）
  • 回饋者可請求匿名（本平台尊重）
  • 若回饋揭露之事實涉法律訴訟可能，本平台先行下架重審
