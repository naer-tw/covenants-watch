PI-16 教科書中的兩公約敘事

摘要

108 課綱（2019 起施行）公民與社會領綱含「人權公約」單元，教科書（康熹、龍騰、翰林、三民等版本）對兩公約之敘事各有取捨。本卡片以內容分析法檢驗教科書呈現之條文範圍與選擇性。

課綱中兩公約相關學習內容

依教育部 107 年公告之社會領域課程綱要（108 學年度起施行），公民與社會由四大主題構成（A 公民身分認同；B 社會生活組織制度；C 社會運作治理參與；D 民主社會理想現實）。

直接相關條目： - 「我國如何將國際人權公約的相關主張融入法律制度中」（必修） - 多元文化社會與「肯認多元、重視人權」素養目標 - 加深加廣選修對人權保障 / 權利衝突之延伸探討

課綱條目以提問式撰寫，未在條目層級指定 ICCPR/ICESCR 各條文之教學細節。

教科書版本之人權單元（公開可見之章節結構）

5 大版本必修第二冊均設有「人權」相關單元： - 社會規範與國際人權 - 民主正當性與權利主體 - 基本權利與憲政主義 - 我國人權保障與國際接軌（兩公約施行法） - 兒少權利 / 性別平等 / 勞動人權專題

「兩公約施行法（2009）」與「世界人權宣言」普遍出現於課文與大事年表。

學測 / 分科測驗考題類型

大考中心 111 學年度分科測驗公民與社會考試說明明示人權為核心評量主題。歷年公開試題可見： - 兩公約施行法之內國法效力辨析 - 司法院釋字 / 憲法法庭判決與 ICCPR 條文對應 - 死刑、集會遊行、言論自由公約檢視題組 - 經社文公約下勞動權、健康權情境題

ICCPR 18-4 父母教育選擇權之呈現（顯性化議程聲明）

本平台特別關注此條文之理由（顯性化聲明）：國教行動聯盟之家長立場使得本平台對 18-4「父母為其子女選擇宗教及道德教育之自由」格外敏感。本節非中立觀察，是平台議程之延伸；讀者應據此自行評估資訊。

為避免單條文盤點之偏頗，本平台亦承諾在 Wave 90+ 對 22 條核心條文做完整缺席率盤點（含 Art.6 死刑、Art.27 原民、ICESCR Art.7 勞動權等），讓 18-4 之缺席率在資料分布中自然浮現。

初步觀察（待 Wave 90+ 完整盤點）： - 公開可檢索之教科書目錄與市售複習講義摘要中，未見以 ICCPR 18-4 父母教育選擇權為獨立子題之章節標題 - 相關內容多歸入「宗教自由」或「兒少權利」之整體框架 - 此非定論——須取得紙本 / 授權電子書才能核實內文呈現比重

學界評論（雙面）

  • 支持方：肯認課綱深化「肯認多元、重視人權」之素養導向
  • 批評方：條目以提問句撰寫導致教學負擔加重、教師對國際人權法專業知能不一、教材詮釋空間大

資料缺口

  1. 5 大版本教科書全文受著作權保護，公開檢索僅能取得目錄／單元名稱，內文敘事比重無法量化
  2. 大考中心未公開「人權題型」分年統計，需逐年逐題人工編碼
  3. ICCPR 18-4 於各版本之實際呈現字數、是否提及父母權，需取得授權電子書方可核實
  4. 學界對課綱人權單元之同儕審查論文有限，多為教學現場心得

二、§Steelman

教科書編者可主張：「篇幅有限，編者必須擇要呈現。選擇反映社會關注度與當代議題重要性。」

此論述部分成立。但： - 「社會關注度」與「條文重要性」不必然對等 - 18-4 父母教育選擇權之低呈現，與 26 條不歧視之高頻出現，兩者法律位階相同 - 若教科書之選擇性與 PI-12 公領域援引選擇性高度相關 → 構成世代再製

三、§Control Group

對照三公約（CRC / CEDAW / CRPD）在教科書中之呈現選擇性，回答：「兩公約被特別選擇性？或四公約皆呈現相同模式？」
