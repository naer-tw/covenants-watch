# 內部研究資料夾

## 此資料夾的用途

此資料夾**不對外發布**。內容包含：
- 研究方向之預設假設（PI 卡片實證前的 H1 草稿）
- 待量化驗證的觀察印象
- 可能存在但尚未證實的偏見記錄
- 內部討論與爭議筆記

## 為何不對外公開

**避免「先射箭再畫靶」**。本平台對外文件只能呈現：
1. 研究設計（H0/H1 雙向）
2. 已完成資料蒐集 + 統計檢定的結果
3. 經外部審查通過的詮釋

對外文件**不可**呈現：
- 預期某條文/政黨/團體會出現某種傾向的假設
- 用「示意」「待補」名義流出的具體數字（如 100 vs 5 等）
- 任何點名特定團體 / 政黨可能「工具化」「避談」的措辭

## 對應 ADR

- ADR-003 立場明確且公開
- ADR-005 對事不對人原則
- ADR-007 援引異常的客觀判準（× 10 改為統計檢定）

## 部署規則

- `internal/` 列入 `.gitignore` 之後不上 git remote（如有 push 需求）
- 但保留本地 git history 供研究團隊內部追蹤
- 跑 `bash scripts/sync_to_jekyll.py` 時必須過濾 `internal/`
- `_public/` 公開頁面任何連結指向 `internal/` 都應視為錯誤
