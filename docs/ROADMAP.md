## 風險登記

| 風險 | 機率 | 衝擊 | 對策 |
|---|---|---|---|
| 政治攻擊（被指為「反同」「反進步」） | 高 | 高 | 客觀證據優先、立場聲明顯眼、提供異議申訴 |
| 法律訴訟（誹謗 / 個資）| 中 | 高 | 不點名個人 + archive.org 典藏 + 72h 回應 |
| 資料源被刪 / 修改 | 中 | 中 | 全部 archive 典藏 + git 歷史保存 |
| 內部一致性破口 | 中 | 中 | 每 5-10 wave cold read QA + self_qa.sh |
| 倡議夥伴退出 | 中 | 中 | 維持平台中立性、不專屬單一 NGO |
