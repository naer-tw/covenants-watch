# Youth Advisory Group SOP — 兒少顧問小組標準作業程序

> 對應 MVP 藍圖 §12 + governance/child_safeguarding.md + 兒童權利公約 GC12 兒少表意權
> 設立目的:讓平台之政策建議由兒少實質審議過(not just consulted),符合 CRC Art. 12 之精神

## 一、定位

**外部諮詢機構**(非組織內部成員)。與**國教盟青年部**之差別:
- 青年部 = 國教盟組織內部之青年參與架構(已存在,辦過 8,743 份問卷)
- 兒少顧問小組 = 平台**對外開放**之獨立諮詢角色,不需是國教盟成員

兩者**互補不重疊**:青年部主導大規模問卷與議題擴散;顧問小組負責**深度審議**(影子報告兒少版校稿、議題卡片白話化、政策建議可行性檢核)。

## 二、組成

| 組成 | 名額 | 比例 | 任期 |
|---|---:|---|---|
| 國中代表 | 4 名 | 50% | 1 年(可續任 1 次)|
| 高中代表 | 4 名 | 50% | 同上 |
| **總計** | **8 名** | | |

**多元性要求**:
- 性別:每組至少 2 種性別表達
- 地區:都會 / 偏鄉至少各 1 名
- 學校型態:一般 / 實驗 / 特殊教育至少有代表
- 經歷:有人權倡議經驗者優先(青年部、模聯、社團領袖等)

## 三、招募流程(每年 9 月開放,12 月確定名單)

```
第 1 步:9 月開放申請(平台 _public/kids/recruit.html)
  ├─ 自薦(13 歲以上自主同意)
  ├─ 青年部推薦
  └─ 學校 / NGO 推薦

第 2 步:書面審查(主編 + 兒少保護專員)
  ├─ 動機說明 ≤ 500 字(白話)
  ├─ 同意書(本人 + 13 歲以下加家長簽名)
  └─ governance/child_safeguarding.md §三點五 倫理確認

第 3 步:面談(視訊 30 分鐘)
  ├─ 主編 + 1 名青年部代表 + 1 名外部 NGO 顧問
  └─ 重點:傾聽風格 vs 主導風格

第 4 步:出席選議題會議(2 次)觀察
  └─ 評估能否在 4 小時會議中保持有效參與

第 5 步:12 月公告新一屆 + 任期銜接會
```

## 四、會議節奏

### 4.1 季會(每年 4 次,線上為主)

每次 90-120 分鐘,議程固定:
1. **0-15 分鐘**:本季新議題介紹(主編)
2. **15-45 分鐘**:議題深度討論(顧問發言為主)
3. **45-60 分鐘**:政策建議草稿審議(逐條表決可 / 不可)
4. **60-75 分鐘**:回饋上季決議之執行成效(Lundy 影響力閉環)
5. **75-90 分鐘**:自由提案(顧問可帶入身邊看到的新議題)

### 4.2 影子報告校稿週(每年 9-10 月,1 次,實體優先)

CRC 審查週期或重大政策節點時,集中校稿:
- 兒少版 12 cluster 全部過一遍
- 提名 1-2 名顧問作為「兒少版主筆」
- 紀錄改寫前後文字對照,作為平台白話化標準

### 4.3 緊急會議(突發事件)

如剴剴案、新北割頸案等重大事件發生時,72 小時內召開緊急視訊:
- 顧問可選擇 opt-in 參與(非強制,避免再創傷)
- 平台主編簡報事件 + 法律觀點 + 媒體應對草稿
- 顧問提供「兒少視角」之發言或聲明素材

## 五、議事規則(對應 Lundy Model)

| Lundy 步驟 | 平台實作 |
|---|---|
| **Space**(安全空間) | 視訊有人陪、有 break、有「pass 不發言」選項 |
| **Voice**(實質發聲) | 顧問每人至少 5 分鐘專屬時間,主編不打斷 |
| **Audience**(對的決策者) | 議程外另設「上呈管道」— 平台會把意見帶到立委/部會回應 |
| **Influence**(影響力閉環) | 每季回顧上季顧問建議**有沒有被採納 / 被誰採納 / 後續發展** |

## 六、與青年部之分工

| 工作 | 青年部 | 顧問小組 |
|---|---|---|
| 大規模問卷 | ✅ 主辦 | 提供題目意見 |
| 影子報告兒少版校稿 | 提供意見 | ✅ 主審 |
| 議題卡片白話化 | 提供初稿 | ✅ 校對 |
| 政策建議可行性檢核 | 提供觀點 | ✅ 投票 |
| 媒體聲明草擬 | 提供素材 | 補充兒少視角 |
| 公聽會發言 | ✅ 出席 | 旁聽(15-18 歲可發言)|
| 平台介面測試 | ✅ 主辦 | 提供回饋 |

## 七、薪酬與認證

- **車馬費 / 通訊費**:每場會議 NT$ 800(線上)/ NT$ 1,500(實體 + 含交通)
- **獎狀 / 證書**:每年由理事長頒發「兒少權利倡議顧問」證書(寫進大學申請推薦信可用)
- **公開致謝**:顧問可選擇是否在影子報告致謝段落具名(預設匿名)

## 八、退出機制

- **隨時可退出**:寫信說明即可,無理由要求
- **強制離職**:洩漏其他顧問身分 / 違反 governance/child_safeguarding §三點五 / 連續缺席 3 次無故
- **離職後保密期**:1 年內不公開其他顧問之發言內容(除非經對方同意)

## 九、與既有治理之關聯

| 既有文件 | 本檔之關聯 |
|---|---|
| `child_safeguarding.md` | 顧問小組成員為兒少,所有討論之發言皆受 §三點五 保護 |
| `transcript_citation_sop.md` | 會議紀錄錄音僅供主編使用,不對外;摘要可公開但去識別 |
| `proxy_kid_version_disclosure.md` | 顧問小組校稿後之兒少版,不再標記「成人代擬」 |
| `privacy_policy.md` | 顧問個資加密存放,未經本人同意不對外 |
| `rbac.md` | 顧問擁有「外部專家審閱」+ 「兒少版發布」之權限 |
| `document_governance.md` | 顧問校稿之文件升至 `publish` 層需經主編 + 兒少保護專員雙簽 |

## 十、年度報告

每年 12 月平台須公布:
- 顧問小組組成統計(去識別化)
- 季會召開次數 + 平均出席率
- 顧問建議被採納之具體案例(至少 5 件)
- 顧問離職率 + 原因類別
- 來年改善方向

## 十一、本檔之啟動條件(致用戶端)

本 SOP 為**就緒文件**,實際啟動需:
1. ☐ 完成第 1 屆招募(9 月公告 → 12 月確定)— 預計 2026-09 ~ 2026-12
2. ☐ 確定平台主編 + 兒少保護專員 + 外部 NGO 顧問之具體人選(governance/INDEX.md §3 角色任命)
3. ☐ 取得 NT$ 80,000 ~ 120,000 年度預算(8 人 × 4 季 × NT$ 800-1,500 + 證書印製 + 緊急會議準備)
4. ☐ 平台主站建立 `_public/kids/recruit.html` 招募頁面(本 wave 不做,Wave 80+)
