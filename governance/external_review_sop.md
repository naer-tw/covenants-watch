# 外部審查邀請手冊 + 法律顧問複核 SOP

> 第四輪 cold read QA 結論：「不建議現在上線」，須先完成本流程。
> 本手冊為平台正式 v1.0 上線前之必要程序。

## 一、外部審查委員邀請（3 位）

### 邀請對象（依 _STEELMAN_AND_CONTROL_GROUPS.md SOP-3）

#### A. 研究方法論專業者（1 位）
- **背景**：統計學或公共政策博士
- **責任**：審查 PI-12 / PI-13 之 chi-square / Gini 統計設計、樣本範圍、預先註冊機制
- **建議邀請**：政大選研中心、中研院政治所、台大公衛所

#### B. 法律學者（1 位）
- **背景**：人權法 / 憲法 / 國際法
- **責任**：審查 PI-04 兩公約施行法詮釋、PI-10 詮釋光譜法源、Steelman 雙面強度
- **建議邀請**：台大法學院（黃昭元）、中研院法律所、東吳法學院

#### C. 反對本平台立場之學者（1 位，**必要邀請**）
- **背景**：性別研究 / LGBT 研究 / 公衛政策 / 教育學
- **責任**：對本平台之 PI-10 / PI-16 / 整體立場做最強反論審查
- **目的**：確保平台不淪為單面文宣
- **建議邀請**：台大女性研究中心、政大性別研究所、台灣性別人權協會學者

## 二、法律顧問複核 SOP

### 複核重點議題卡

依風險強度排序：

#### 高風險（必須複核才能對外發布）
- **PI-09**：NCC 處分案件量化（個別案號 + GC34 三要件對照）
  - 須匿名編號 + 案件類型 + 處分金額區間
  - 個別當事媒體 14 日異議期
- **PI-13**：立法院公報援引案例（即使已移除政黨欄）
  - 公報為公開資料，但對個別法案做歸因敘事仍須法律確認
- **PI-15**：人權白皮書 vs 預算 vs 法案 vs 民意四維對照
  - 立委個人提案之引用避免名譽風險

#### 中風險
- **PI-02**：13 年來「abandoned/pending/done/partial」分類（民間觀點）
  - 須確認分類描述非「政府失職」之主觀指控
- **PI-14**：民間代表團組成（NGO 議題分布）
  - NGO 立場分類須附 14 日異議期
- **PI-08**：太極門案個案分析
  - 已被法院判決公開，但仍須法律確認引用範圍

#### 低風險
- 其他 PI 主要為文獻整理，風險較低

### 法律顧問建議流程

```
Step 1 (Week 1)：法律顧問通讀全 16 PI + 影子報告
Step 2 (Week 2)：高風險 PI 之逐條條文檢視
Step 3 (Week 3)：建議修改清單 → 平台修補
Step 4 (Week 4)：複審 → 簽署「上線無顯著名譽風險」同意書

費用估計：NT$ 80,000-120,000（含 4 週工作 + 簽署證明）
```

### 法律顧問選擇

- 具誹謗罪 / 名譽侵害訴訟經驗
- 對人權公約有基本認識
- 不與平台之主要立場有利益衝突
- 建議：理律法律事務所、寰瀛、知名人權律師事務所
