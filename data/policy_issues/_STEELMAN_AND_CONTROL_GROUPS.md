# Steelman 與對照組治理規範

## 一、本規範解決的問題

cold read QA 指出：若本平台僅呈現「兩公約被工具化」一面論述，而不呈現倡議方對 Art.26 / Art.23 詮釋的完整理據，會被定位為**單面文宣**而非客觀研究平台。

此外，僅針對兩公約進行集中度分析，無法證明「兩公約特別被工具化」——須對照其他施行法（CRC / CEDAW / CRPD）才能站住結論。

## 二、強制 SOP

### SOP-1：每張 PI 卡必須含 §steelman 段落

任何 PI 卡片在發布實證結論前，必須包含 §steelman 段落，內容：
1. 完整呈現本平台**反對立場**之最強論述（不弱化、不簡化）
2. 引用對方最具代表性的 1-3 篇學術 / 倡議文獻
3. 明確指出對方論述的**最強證據**（即便對本平台不利）
4. 才能進入本平台的反論

### SOP-2：CRC / CEDAW / CRPD 對照組必跑

任何「兩公約 X 條集中度 / 援引偏差」之量化研究結論，必須同步：
- 對 CRC（兒童權利公約）跑相同手法 → 取得集中度基準
- 對 CEDAW（消除對婦女歧視公約）跑 → 取得性別議題敏感度基準
- 對 CRPD（身心障礙者權利公約）跑 → 取得身障議題基準

只有當**兩公約的偏差 > 三個對照組的偏差中位數**時，才能下「兩公約被特別工具化」結論。

否則只能下「人權施行法之援引普遍呈現特定模式」結論，不能單獨歸因兩公約。

### SOP-3：外部審查委員制度

發布任何實證結論前，須邀請以下三位審查者盲審：
- 1 位具研究方法論專業（統計或公共政策博士）
- 1 位具法律學者背景（人權法 / 憲法）
- 1 位具**反對本平台立場**的學者（須明示）

審查意見須於發布頁面公開摘要。

### SOP-4：自我問責機制

設立外部問責委員會（3-5 人，至少 1 位為非贊成方）：
- 每季抽查 5% 對外文章是否符合本平台紅線
- 接受民眾異議申訴 + 30 日內公開回覆
- 申訴成立則公開訂正，並記錄於 ADR

## 三、適用對象

本 SOP 對所有 D 區（工具化證據）PI 卡 **強制適用**：
- PI-12 條文援引頻率
- PI-13 立法院公報量化
- PI-14 民間代表團組成
- PI-15 政府人權白皮書 vs 施政落差
- PI-16 教科書中的兩公約敘事

對 A/B/C 區 PI 卡**選擇性適用**：
- 若該 PI 卡含具體比較或排序時 → 適用
- 若僅為資料整理時 → 可豁免

## 四、不遵守的後果

任何 PI 卡未通過 SOP-1/2/3 但對外發布的，視為平台紅線違反，須立即下架重審。
