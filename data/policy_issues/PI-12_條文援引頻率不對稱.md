# PI-12 兩公約條文援引頻率分布之客觀檢驗

## 摘要

兩公約共 22 條核心實質權利條文（ICCPR Art.6-27 / ICESCR Art.6-15），是否在台灣公領域被**對等援引**？或某些條文被高頻使用、其他條文被忽視？

本卡片**僅提出研究設計與資料蒐集計畫**，不在實證資料蒐集完畢前下任何具體結論。**所有「預期方向」「假設立場」皆移至內部研究文件 `internal/PI-12_hypotheses.md`，不對外公開。**

> **本卡片的對外承諾**：在 Wave 21+ 實證資料完整匯入並通過外部 QA 前，本卡片不發布任何條文間援引比較數字、不點名任何政黨或團體。

## 一、研究設計

### 樣本範圍
1. **立法院公報**：2009-12-10（兩公約施行法施行）至最新，全文檢索
2. **主流媒體投書與報導**：以發行量與穩定 archive 為標準選 6-8 家媒體（清單於資料蒐集前公開）
3. **NGO 倡議文件**：以「公開出版且 archive.org 已典藏」為唯一標準，不依立場篩選；清單於資料蒐集前公開
4. **政府文件**：行政院公報、各部會新聞稿、4 次國家人權報告、2 期 NAP

### 度量指標
- 每條條文被明確援引次數（標出 ICCPR Art.X 或 ICESCR Art.Y）
- 條文 × 來源類別 × 年度交叉表
- 政黨／NGO／媒體欄位**分組編碼後分析**（以政黨代碼 P1/P2/P3、NGO 代碼 N01-N99 等），論文層級才解碼

### 研究問題（開放式）

1. 22 條核心條文之援引頻率分布為何？
2. 援引頻率是否與條文「重要性」（依 NHRC 指標 / 國際 GC 指引）對等？
3. 不同來源類別（政黨、NGO、媒體）的援引偏好是否有顯著差異？
4. 14 年來的援引演變是否呈現特定時點的政策偏移？

> 本平台**不預設**任一條文「應該」被援引多少次，僅以資料說話。

### 統計判準

- 採卡方檢定（chi-square test）測量條文間援引分布是否顯著偏離均勻分布
- 採基尼係數（Gini coefficient）量化集中程度
- 與其他國際公約施行法（CRC / CEDAW / CRPD）相同手法計算後**比較三者集中度**，避免單獨比較失真
- 顯著閾值預先設定為 p < 0.01，並全程公開原始資料供獨立重現

## 二、資料蒐集 SOP

```bash
# 1. 立法院公報下載 + 全文索引
python3 scripts/two_cov_fetch_legislative.py --years 2009-2026

# 2. 條文援引計次（ICCPR Art.X 或 ICESCR Art.Y 字串比對）
python3 scripts/two_cov_count_citations.py \
  --source legislative \
  --output data/evidence/citation_legislative.csv

# 3. 媒體報導 archive 蒐集
python3 scripts/two_cov_fetch_media.py --media-list deployment/media_targets.yaml

# 4. 統計分析
python3 scripts/two_cov_statistical_analysis.py \
  --tests chi_square,gini \
  --output _public/dashboards/citation_distribution.html
```

> 上述腳本將於 Wave 21+ 實作。所有腳本將以 BSD-3 開源，第三方可獨立執行重現結果。

## 三、品質控管

1. **預先註冊**：研究設計於資料蒐集**前**公開於本卡片，事後不得修改度量指標（避免 p-hacking）
2. **原始資料公開**：所有抓取結果以 SQLite + CSV 雙格式公開
3. **獨立重現**：第三方可重跑全部腳本得到相同結果
4. **外部審查**：在發布結果前，邀請至少 1 位研究方法論專家、1 位法律學者、1 位倡議運動立場學者進行盲審
5. **異議申訴**：被認為「不對稱」之團體 / 政黨可在報告發布後 30 日內提出方法論挑戰，若成立則修訂結論
